hsumhres_tb.tcl : Tcl script to run in Modelsim/Questasim to run hsumhres_tb
                  testbench with all possible generics.

hsumddrin_tb.tcl : Tcl script to run in Modelsim/Questasim to run hsumddrin_tb
                  testbench with all possible generics.

compile.tcl : Tcl script to compile all HSUM VHDL source and testbench files.
              Note: Requires to be run in windows tclsh. Check shebang when running
                    from Cygwin.
